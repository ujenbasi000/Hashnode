# Hashnode Clone

This is a simple clone of a popular developer blogging site called [Hashnode](https://hashnode.com).

## Featues:

- Authentication
- Crud operation of blog post
- Crud operation of user
- Responsive design
- Image upload using cloudinary
- Creating Tags
- Follow Tags, Users
- Search by username, tags, title
- Like, Comment Post
- Save blog for future use
- Auto save in writing area
- Infinite Scoll
- Markdown
- Light / Dark Theme

## Tech Stack:

- Nextjs / Reactjs
- Mongodb
- Cloudinary
- Apollo Graphql
- Express / Node
- Jsonwebtoken
- Tailwindcss
